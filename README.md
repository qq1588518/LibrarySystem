# Library-Management-System
## 系统
基于C#语言和sql server2008开发

## 功能
* 图书信息
    *  图书基本信息的`新增`
    *  图书基本信息的`修改`
    *  图书基本信息的`删除`
    *  图书基本信息的`查询`

* 读者信息
    *  读者基本信息的`入库`
    *  读者基本信息的`修改`
    *  读者基本信息的`删除`
    *  读者基本信息的`查询`

* 借阅信息
    * 读者借阅
    * 读者续借
    * 查询借阅信息

* 罚款
    * 超期罚款
    * 丢失罚款
    * 损坏罚款

* 统计信息

* 搜索
